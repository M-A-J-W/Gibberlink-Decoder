# Audio Recorder & Decoder

# Code gerne benutzen

Browserbasiertes Tool zum **Aufnehmen und Abspielen von Audio** mit Live-Fortschritt. Unterstützt **Drag & Drop**, Datei-Upload und Aufnahme in **WebM** oder **WAV**.

## Funktionen

* **Audio abspielen** mit Fortschrittsanzeige
* **Audio aufnehmen** direkt im Browser (WebM/WAV).
* **Drag & Drop** oder klassische Dateiauswahl.
* **Abbrechen** von Wiedergabe oder Aufnahme per Button.

## Nutzung

1. Dateien per Drag & Drop oder Datei-Input hinzufügen → Wiedergabe startet automatisch.
2. Aufnahme starten → Fortschritt sichtbar, nach Beendigung URL in Konsole.
3. Abbrechen jederzeit möglich.

## Anforderungen

* Moderner Browser mit `AudioContext`, `MediaRecorder` und `getUserMedia`.
* Keine zusätzlichen Bibliotheken nötig.
